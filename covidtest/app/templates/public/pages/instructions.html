{% blocktrans %}Anleitung Probennahme{% endblocktrans %}

{% blocktrans %}Die Teststation analysiert Gurgelproben auf das Vorhandensein des Corona-Virusgenoms. Zur Entnahme der Gurgelprobe benutzen Sie bitte das Testkit, bestehend aus einer Plastikflasche mit steriler Kochsalzlösung, einem Strohhalm, einem Teströhrchen mit Barcode sowie einem kleinen Beleg.{% endblocktrans %}

{% blocktrans %}So funktioniert die Gurgelprobe:{% endblocktrans %}

  • {% blocktrans %}Bitte noch VOR dem Frühstück oder erst mindestens 30 Minuten nach dem Essen/Zähneputzen die Gurgelprobe abnehmen. Halten Sie eine Schere bereit.{% endblocktrans %}
  • {% blocktrans %}Schrauben Sie die Plastikflasche auf und verwenden Sie die gesamte 5ml-Kochsalzlösung zum Gurgeln tief in der Kehle (nicht schlucken!).{% endblocktrans %}
  • {% blocktrans %}Gurgeln Sie mit der Kochsalzlösung für 30s und spülen Sie sich auch den Mund damit aus.{% endblocktrans %}
  • {% blocktrans %}Füllen Sie die Gurgelflüssigkeit mit dem Strohhalm in die Plastikflasche.{% endblocktrans %}
  • {% blocktrans %}Schütteln Sie die Plastikflasche 30s lang, um die Probe zu gut zu durchmischen.{% endblocktrans %}
  • {% blocktrans %}Schneiden Sie die Spitze der Plastikflasche mit der Schere ab und füllen Sie das mitgelieferte Röhrchen mit Barcode nahezu voll. Achten Sie darauf, dass Sie das Röhrchen richtig verschließen.{% endblocktrans %}

{% blocktrans %}Probenabgabe:{% endblocktrans %}

  • {% blocktrans %}Überprüfen Sie sorgfältig, ob der Tube code auf dem Beleg mit dem Barcode auf den Teströhrchen mit Ihrer Probe übereinstimmt.{% endblocktrans %}
  • {% blocktrans %}Werfen Sie das Teströhrchen mit Ihrer Probe in einen der Sammelbriefkästen auf dem Campus.{% endblocktrans %}
  • {% blocktrans %}Bewahren Sie den Beleg mit dem Zugangscode an einem sicheren Ort auf und fragen Sie das Ergebnis am Abend oder am nächsten Tag auf dieser Webseite ab.{% endblocktrans %}
  • {% blocktrans %}Zur Erinnerung können Sie Ihren Namen und das Datum auf dem Beleg vermerken.{% endblocktrans %}
